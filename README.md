14-2 科技華語 - 課程作業展示網頁

這是一個靜態網頁專案，專門用來展示「114-2 科技華語」課程中，學生們在各個單元的學習成果與創意提案。本專案目前收錄了共 8 次作業的入口，並已完整建置「第一次作業（Unit 1）」的內容。

🌟 專案特色

無伺服器架構 (Serverless)：完全由前端技術打造，無需安裝 Node.js 或任何後端環境即可運行。

響應式設計 (RWD)：採用 Tailwind CSS 進行切版，完美支援手機、平板與電腦螢幕。

現代化 UI/UX：使用清新的薄荷綠與藍灰配色，搭配流暢的淡入動畫與精緻的卡片排版。

React 元件化開發：透過 CDN 引入 React 與 Babel，直接在 HTML 檔案中撰寫 JSX，方便維護與擴充。

📂 檔案結構

index.html：課程首頁。包含課程介紹以及 8 次作業的導覽入口（目前 Unit 1 已解鎖）。

unit1.html：第一次作業（Unit 1）專屬頁面。展示學生們針對「校園生活痛點」所設計的創新 AI 小工具提案（包含痛點分析、產品功能、風險評估與定價推廣）。

🛠️ 使用技術

HTML5

Tailwind CSS (via CDN)

🎓 關於 Unit 1：校園 AI 助手提案

本單元透過 PBL (Project-Based Learning) 專題任務，引導學生觀察校園生活痛點，並提出具創意的 AI 解決方案與商業模式。提案遵循四大核心指導原則：

痛點分析 (Pain Point)

產品功能 (Solution)

風險評估 (Risk)

定價推廣 (Business)
